ШАГ 1: УДАЛИТЕ СТАРЫЕ ФАЙЛЫ

1. Откройте https://github.com/malinka-lab/neiro-key/settings
2. Прокрутите вниз до "Danger Zone"
3. Нажмите "Delete this repository"
4. Введите: malinka-lab/neiro-key
5. Подтвердите удаление

═══════════════════════════════════════════════════════════════

ШАГ 2: СОЗДАЙТЕ НОВЫЙ РЕПОЗИТОРИЙ

1. Откройте https://github.com/new
2. Repository name: neiro-key
3. Выберите: Public
4. НЕ добавляйте README
5. Нажмите "Create repository"

═══════════════════════════════════════════════════════════════

ШАГ 3: ЗАГРУЗИТЕ ФАЙЛЫ

1. В новом репозитории нажмите "uploading an existing file"
   
2. Перетащите ВСЕ файлы из этой папки:
   ✓ index.html
   ✓ папку assets
   ✓ logo.png
   ✓ photo1.jpg
   ✓ photo2.jpg
   ✓ photo3.jpg
   ✓ photo4.jpg
   ✓ photo5.png
   ✓ testimonial.png
   ✓ README.md

3. Нажмите "Commit changes"

═══════════════════════════════════════════════════════════════

ШАГ 4: ВКЛЮЧИТЕ GITHUB PAGES

1. Settings → Pages (в левом меню)
2. Source: Branch: main, Folder: / (root)
3. Нажмите "Save"
4. Подождите 2-3 минуты

═══════════════════════════════════════════════════════════════

ШАГ 5: ОТКРОЙТЕ САЙТ

Ваш сайт будет доступен по адресу:
https://malinka-lab.github.io/neiro-key/
